1) What is the difference between echo and print commands?

Echo prints whatever strings or string literals you pass it.
Print prints the values and types of the variable you pass it.

7) List String Operators in PHP

  • stra . strb - Concatenation
  • stra .= strb- Appends strb to stra
  • strlen(str) - get length of given str
  • str_word_count(str) - get number of words in str
  • strrev(str) - reverse given string
  • strpos(str, subStr) - returns index number where subStr is located in str
  • str_replace(needle, str, haystack) - replaces needle in haystack with str

8) List Array Operators in PHP.

  • arr1 + arr2 - Union of two arrays
  • == - check Equality and have the same key/value pairs
  • === - check Equality and have samy key/value pairs in same order
  • != or <>- Not Equal
  • !== - Not Identical
  • array(val1, val2, val3, etc...) - array instantiation
  • array(key1 => val1, key2 => val2, key3 => val3, etc...) - array instantiation
  • var[] = val - add value to array
  • var[key] = val - set value of key
  • var[key] - get value of key in array var, where key can be index number or named-key

10) What are the array types in PHP?

  • indexed - Arrays with a numeric index
  • associative - Arrays with named keys
  • multidimensional - Arrays containing one or more arrays

11) What is the difference between Tree-Based XML Parsers and Event-Based XML Parsers?

  • Event-Based Parser - Processes an encountered element and then forgets about it.
  • Tree-Based Parser - An in-memory tree is created for the entire document.
